# 🔮 Astrology & Personality Prediction Website

A frontend-based **Astrology and Personality Prediction Web App** built using **HTML, CSS, and JavaScript**.  
The website takes user details like **name and date of birth** and provides zodiac information, compliments, emotional insights, recommendations, and future predictions.

> A basic idea, but big learning ✨

---

## 🚀 Features

- User input form with:
  - First Name
  - Last Name
  - Date
  - Month
  - Year
- Zodiac sign calculation
- Daily compliment message
- Emotional (victim-card style) message
- Personal improvement recommendations
- Future prediction message
- Randomized & date-based logic
- Clean and responsive UI

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (ES6)** – Logic & interactivity

---

## 🧠 How It Works

1. User submits personal details via form.
2. JavaScript processes the input:
   - Calculates zodiac sign
   - Selects compliment based on date
   - Generates emotional validation message
   - Provides a recommendation for personal growth
   - Predicts a future insight
3. All results are displayed dynamically on the page without reload.
